The Literature Application

The literature application is a simple application that provides a search function to all uploadet literature. The serach function searches on title, author and tags, that may have been included with the literature file during upload. The database is located on google cloud and the data is transferred using Apache HTTP Server. A user login is required to use the application, and only admin users can upload new items.
